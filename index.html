01 / About

Engineer at the intersection of

I'm a Master's student in Engineering Sciences at TH Rosenheim with a background in Mechanical Engineering and hands-on industry experience in embedded systems and model-based development for automotive EV control systems.

My focus is on state estimation, model predictive control, and real-time embedded software — the intersection where rigorous math meets hardware that has to work reliably at 100 km/h.

I learn by building: every concept I study goes into a working prototype, whether that's a Kalman filter running on an STM32 or an MPC controller tuned in simulation.

Battery SOC Estimator on STM32

Equivalent Circuit Model (ECM) built in Simulink, paired with an Extended Kalman Filter implemented first in MATLAB then hand-ported to C. Runs on an STM32F4 Discovery board under FreeRTOS with real-time task scheduling and UART telemetry.

Lateral MPC for Autonomous Vehicle Steering

Model Predictive Controller for lateral vehicle guidance using a bicycle model, implemented with acados and CasADi. Explores constraint handling, state-space linearisation, and real-time feasibility for ADAS steering applications.

Modular EKF Sensor Fusion (2D Navigation)

Plug-and-play Extended Kalman Filter framework for 2D robot navigation in Python simulation. Modular sensor architecture inspired by QuNav, with configurable measurement models for IMU, GPS, and encoder fusion.

About

I'm a Master's student in Engineering Sciences (Mechatronics) at TH Rosenhiem with a background in Mechanical Engineering and hands-on industry experience in ECU application software development for automotive EV control systems.

Now at master's level I'm going deeper: MPC, robust estimation, acados-based solvers, hardware implmentation. The goal is to work at the point where advanced control algorithms meet production embedded constraints.

I learn by building: every concept I study goes into a working prototype, whether that's a Kalman filter running on an STM32 or an MPC controller tuned in simulation.

CSTM32F4FreeRTOSMATLAB/SimulinkEKFECM

BMS needs accurate SOC under noisy measurements — pure Coulomb counting drifts badly over time and temperature.
Approach

PythonacadosCasADiMPCVehicle dynamics
Problem
Lateral control must respect actuator limits simultaneously — classical PID cannot handle constraints explicitly.
Approach
Linearised bicycle model, MPC in acados/CasADi, constraints on lateral error and steering rate.
Result
Solver runs in <[X] ms per step, constraint satisfaction verified across lane-change and curve scenarios.

P-03  ·  Modular EKF Sensor Fusion
PythonEKFNumPyIMU/GPS/Encoder
Problem
Sensor fusion implementations are tightly coupled to a fixed sensor set — swapping sensors means rewriting the estimator.
Approach
Plug-and-play EKF architecture — each sensor registers its own measurement model independently.
Result
Adding a new sensor requires only a measurement Jacobian — core filter logic unchanged. Validated in 2D sim.
View repo ↗
P-04  ·  MPC on Nonlinear CSTR
MATLABMPCNonlinear systems
Problem
CSTR has strong nonlinearity — a single linearisation point fails when the system moves far from it.
Approach
Nonlinear system ID, successive linearisation across operating range, MPC with terminal cost for recursive feasibility.
Result
Stable tracking across setpoint changes where single-point linear MPC diverged.
